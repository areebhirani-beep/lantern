# /daily-sync

Morning briefing for Lantern.

1. `git status` and `git log --oneline -10` — what changed.
2. Health checks: `npm run lint`, `npm run build`, and `/guardrail-audit`. Report pass/fail.
3. Open threads: failing checks, unfinished submissions in `docs/submissions/`, TODO/FIXME in `src/`.
4. Summarize blockers, top 3 priorities, and the next action.
